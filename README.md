# Password strength checker
An AI-powered Password Strength Checker that analyzes user passwords, provides strength scores, and suggests improvements for weak passwords. The system is built in Python, with a dashboard interface, modular feature extraction, and backend support for secure storage.
## Technologies Used
- Python
- Flask / FastAPI for web interface
- HTML, CSS, JavaScript for frontend
- Password analysis algorithms
- Data visualization for dashboard

---

## Features
- Evaluate password strength in real-time
- Score passwords on security criteria (length, complexity, common patterns)
- Provide suggestions for stronger passwords
- Interactive dashboard for analysis and statistics
